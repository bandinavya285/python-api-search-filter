# API Data Fetcher & Search Filter

A lightweight Python Command Line Interface (CLI) application that fetches user data from a remote REST API, parses the JSON payload, and allows users to search and filter results locally in real-time.

## Key Features
* **Live API Integration:** Utilizes the Python `requests` library to handle HTTP GET requests.
* **JSON Parsing:** Robust extraction and translation of nested JSON objects into native Python data structures.
* **Dynamic Search Filter:** Implements case-insensitive client-side filtering based on user input.
* **Clean UI Output:** Formats raw API data into an organized, readable terminal display.

## Technologies Used
* **Language:** Python 3
* **Libraries:** Requests
* **Target API:** JSONPlaceholder (REST API)
